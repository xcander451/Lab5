# Лабораторная работа №5: Поиск последовательных шаблонов (Sequence Pattern Mining)

## Цель работы
Освоить методы поиска последовательных шаблонов (AprioriAll, PrefixSpan), реализовать базовые алгоритмы, применить их к реальным данным и сравнить их эффективность. Дополнительно решены задачи предсказания следующего события и поиска контрастных последовательностей для выделения ценных клиентов.

## Задания
1. **Реализация AprioriAll с нуля**  
   Загрузка транзакций из CSV, преобразование в последовательности клиентов, генерация кандидатов, подсчёт поддержки и поиск всех частых последовательностей. Проверка на синтетическом примере.

2. **Анализ на реальных данных (Online Retail)**
   - Зависимость числа частых последовательностей от минимальной поддержки (5%, 10%, 20%).
   - Сравнение классических ассоциативных правил (Apriori без учёта последовательности) и последовательных шаблонов.
   - Влияние временного окна `max_gap` (например, 7 дней) на набор шаблонов.
   - Визуализация одного интересного шаблона в виде направленного графа переходов товаров.

3. **Сравнение алгоритмов SPM**  
   Запуск готовой реализации PrefixSpan, замер времени и количества шаблонов, сравнение с собственной реализацией AprioriAll.

4. **Предсказание следующего события**  
   Построение простой рекомендательной модели на основе частых последовательностей длины ≥2. Оценка точности `accuracy` и сравнение с рекомендацией самого популярного товара.

5. **Контрастные последовательности**  
   Выделение групп клиентов (HighValue / LowValue) по среднему чеку. Поиск последовательностей, характерных для каждой группы, и отбор топ-5 контрастных шаблонов. Визуализация и бизнес-интерпретация.
